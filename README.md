# esubdemo

## Overview

`demo_py_esub` is a demo project to illustrate how to organize analysis scripts using Python in a recommended folder structure.

The demo project follows the concepts discussed in:

- [Python for Clinical Study Reports and Submission](https://pycsr.org)

## Folder Structure

By using the recommended folder structure and the associated development tools,
we are able to achieve:

- Consistency
- Automation
- Reproducibility
- Compliance
